Το αρχείο που παραδόθηκε είναι όλος ο φάκελος project_3. Όλη η δουλειά βρίσκεται στο main.java.


Η εργασία γράφτηκε με βάση τον κώδικα της δεύτερης εργασίας. Το makefile είναι το ίδιο με αυτό που δόθηκε στο φροντιστήριο, οπότε μπορούν οι διορθωτές να περιμένουν ότι θα τρέχει με τον ίδιο τρόπο.


Η εργασία υλοποιήθηκε με 2 visitors. Ο πρώτος γεμίζει το symbol map και ο δεύτερος κάνει τα απαραίτητα prints.


Ο symbol map είναι της μορφής <String,<String,String>> όπου το πρώτο string αντιστοιχεί στο εκάστοτε scope(class, method) και ο map φυλάει τις μεταβλητές με τον τύπο τους.


Τα methods φυλάσσονται με την εξής μορφή: Method Class <class_name> <method_name>, ενώ το Class: Class <class_name>.


Όλοι οι arrays (boolean,int) είναι τύπου i32 και κάθε φορά γίνονται τα κατάλληλα bitcast για τις boolean.


Η main δημιουργεί τα αρχεία στον κατάλογο που τρέχει.


Η εργασία φαίνεται να δουλεύει σωστά για όλα τα παραδείγματα που δόθηκαν.


Όταν το αρχείο δεν περνάει τον σημασιολογικό έλεγχο, η main βγάζει στο stdout ένα μήνυμα ότι απέτυχε αυτό το αρχείο και μετά συνεχίζει στο επόμενο. Εκτός αν δοθεί λάθος όνομα αρχείου, όπου και σταματάει ακαριαία.


Πολλές διαγνωστικές print έχουν ξεμείνει ως σχόλια στον κώδικα μήπως χρειαστεί κάποια διόρθωση της τελευταίας στιγμής.
